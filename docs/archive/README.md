# Archive

Historical files from the RatioLab development process. These are preserved for reference but are **not part of the active codebase**.

| File | Description |
|------|-------------|
| `MODEL_HANDOFF.md` | Post-refactor handoff notes from the monolith-to-modular migration. Contains bug hotspots, regression areas, and technical debt inventory. |
| `REFACTOR_PLAN.md` | Original 900-line plan for decomposing the single-file prototype into the current modular architecture. Includes the `compute()` return shape contract, dependency graph, and risk assessment. |
| `servo-scaling-calculator_JSX_OG.txt` | Complete snapshot of the original monolithic `App.jsx` (~834 lines) before modularization. Useful for archaeology and formula verification. |
